Synapses

A number of synaptic ComponentTypes for use in NeuroML 2 documents. These generally extend the baseSynapse ComponentType.
Original LEMS definitions: Synapses.xml

 
baseSynapse

    extends baseVoltageDepPointCurrent
Base type for all synapses, i.e. ComponentTypes which produce a current (dimension current) and change behavior in response to an incoming event.

cno_0000009
Exposures i    (from basePointCurrent) current
Requirements v    (from baseVoltageDepPointCurrent) voltage
Event Ports in Direction: in

baseConductanceBasedSynapse

    extends baseSynapse
Synapse model which exposes a conductance g in addition to producing a current. Not necessarily ohmic!!

cno_0000027
Parameters gbase
    Baseline conductance, generally the maximum conductance following a single spike		 conductance
erev
    Reversal potential of the synapse		 voltage
Exposures i    (from basePointCurrent) current
g
    Time varying conductance through the synapse		 conductance
Requirements v    (from baseVoltageDepPointCurrent) voltage
Event Ports in    (from baseSynapse) Direction: in
Behavior Derived Variables

    i = 0     (exposed as i)
    g = 0     (exposed as g)

 
expOneSynapse

    extends baseConductanceBasedSynapse
Ohmic synapse model whose conductance rises instantaneously by gbase on receiving an event, and which decays exponentially to zero with time course tauDecay
Parameters gbase    (from baseConductanceBasedSynapse) conductance
erev    (from baseConductanceBasedSynapse) voltage
tauDecay
    Time course of decay		 time
Child elements notes notes
Exposures g    (from baseConductanceBasedSynapse) conductance
i    (from basePointCurrent) current
Requirements v    (from baseVoltageDepPointCurrent) voltage
Event Ports in    (from baseSynapse) Direction: in
Behavior State Variables

    g    conductance (exposed as g)

On Start

    g = 0

On Events

    EVENT IN on port: in
        g = g + gbase

Derived Variables

    i = g * (erev - v)     (exposed as i)

Time Derivatives

    d g /dt = -g / tauDecay
 
expTwoSynapse

    extends baseConductanceBasedSynapse
Ohmic synapse model whose conductance waveform on receiving an event has a rise time of tauRise and a decay time of tauDecay. Max conductance reached during this time (assuming zero conductance before) is gbase.
Parameters gbase    (from baseConductanceBasedSynapse) conductance
erev    (from baseConductanceBasedSynapse) voltage
tauRise time
tauDecay time
Child elements notes notes
Exposures g    (from baseConductanceBasedSynapse) conductance
i    (from basePointCurrent) current
Requirements v    (from baseVoltageDepPointCurrent) voltage
Event Ports in    (from baseSynapse) Direction: in
Behavior State Variables

    A    Dimensionless
    B    Dimensionless

On Start

    A = 0
    B = 0

On Events

    EVENT IN on port: in
        A = A + A_factor
        B = B + factor

Derived Variables

    peakTime = ln(tauDecay / tauRise) * (tauRise * tauDecay)/(tauDecay - tauRise)     
    factor = 1 / (-exp(-peakTime / tauRise) + exp(-peakTime / tauDecay))    
    A_rate = -1 / tauRise    
    A_factor = factor    
    g = gbase * (B - A)     (exposed as g)
    i = g * (erev - v)     (exposed as i)

Time Derivatives

    d A /dt = A * A_rate
    d B /dt = -B / tauDecay

scalingConc concentration

Text fields species

Requirements v voltage

nmdaSynapse

    extends expTwoSynapse
NMDA receptor mediated synapse, whose conductance is scaled by a factor dependent on the concentration of magnesium (assumed fixed) and membrane potential
Parameters gbase    (from baseConductanceBasedSynapse) conductance
erev    (from baseConductanceBasedSynapse) voltage
tauRise    (from expTwoSynapse) time
tauDecay    (from expTwoSynapse) time
Child elements notes notes
voltageConcDepBlock voltageConcDepBlock
Exposures g    (from baseConductanceBasedSynapse) conductance
i    (from basePointCurrent) current
Requirements v    (from baseVoltageDepPointCurrent) voltage
Event Ports in    (from baseSynapse) Direction: in
Behavior State Variables

    A    Dimensionless
    B    Dimensionless

On Start

    A = 0
    B = 0

On Events

    EVENT IN on port: in
        A = A + factor
        B = B + factor

Derived Variables

    peakTime = (tauRise * tauDecay)/(tauDecay - tauRise) * ln(tauDecay / tauRise)    
    factor = 1 / (-exp(-peakTime / tauRise) + exp(-peakTime / tauDecay))    
    block = voltageConcDepBlock/scaling    
    g = gbase * block * (B - A)     (exposed as g)
    i = g * (erev - v)     (exposed as i)

Time Derivatives

    d A /dt = -A / tauRise
    d B /dt = -B / tauDecay

stpSynapse

    extends expTwoSynapse
Short term plasticity model based on Tsodyks, M., Uziel, A., and Markram, H. (2000). Synchrony generation in recurrent networks with frequency-dependent synapses. J Neurosci 20, RC50. and Tsodyks, M.V., and Markram, H. (1997). The neural code between neocortical pyramidal neurons depends on neurotransmitter release probability. Proc Natl Acad Sci U S A 94, 719-723.

cno_0000188
Parameters gbase    (from baseConductanceBasedSynapse) conductance
erev    (from baseConductanceBasedSynapse) voltage
tauRise    (from expTwoSynapse) time
tauDecay    (from expTwoSynapse) time
Child elements stpMechanism stpMechanism
Constants tsinceRate = 1 Dimensionless
longTime = 1000s time
Exposures g    (from baseConductanceBasedSynapse) conductance
i    (from basePointCurrent) current
tsince time
R Dimensionless
U Dimensionless
Requirements v    (from baseVoltageDepPointCurrent) voltage
Event Ports in    (from baseSynapse) Direction: in
Behavior State Variables

    A    Dimensionless
    B    Dimensionless
    R    Dimensionless (exposed as R)
    U    Dimensionless (exposed as U)
    tsince    time (exposed as tsince)

On Start

    A = 0
    B = 0
    R = 1
    tsince = longTime

On Conditions

    IF t = 0 THEN
        U = initReleaseProb

On Events

    EVENT IN on port: in
        A = A + factor
        B = B + factor
        R = R * (1-U) * exp(-1 * (tsince/tauRec)) + 1 - exp(-1 * (tsince / tauRec))
        U = U * exp(-(tsince) / tauFac) + initReleaseProb * (1 - (U * exp(-(tsince) / tauFac)))
        tsince = 0

Derived Variables

    peakTime = (tauRise * tauDecay)/(tauDecay - tauRise) * ln(tauDecay / tauRise)    
    factor = 1 / (-exp(-peakTime / tauRise) + exp(-peakTime / tauDecay))    
    initReleaseProb = stpMechanism/initReleaseProb    
    tauRec = stpMechanism/tauRec    
    tauFac = stpMechanism/tauFac    
    plasticityFactor = R * U    
    g = gbase * plasticityFactor * (B - A)     (exposed as g)
    i = g * (erev - v)     (exposed as i)

Time Derivatives

    d A /dt = -A / tauRise
    d B /dt = -B / tauDecay
    d tsince /dt = tsinceRate
